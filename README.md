Описание:

	Это автоматизированный плагин, предназначенный для работы по принципу «перетащи, забудь»
	И обработки всех пользовательских файлов на сервере.
	Плагин предварительно кэширует модели и звуки и добавляет все необходимые файлы в список загрузок.
	Этот плагин также обрабатывает подпапки «custom» для игр и владельцев серверов, которые её используют.

Алгоритм работы:

	1. Сканирование файловой системы при старте карты
	2. Рекурсивный обход папок materials/ models/ sound/
	3. Фильтрация файлов по расширениям (.vmt, .vtf, .mdl, .wav, и т.д.)
	4. Проверка исключений через конфигурационный файл
	5. Добавление в загрузки через AddFileToDownloadsTable()
	6. Предварительное кэширование через PrecacheModel(), PrecacheSound(), PrecacheDecal()
	7. Обработка FastDL - игнорирование .bz2 файлов на сервере (движок делает это автоматически)


Комманды для админа:

	sm_generateexternals или sm_ge генерирует список всех файлов (только для ROOT), посмотреть можно в папке /logs/


Базовое использование:

	Поместите свои файлы в соответствующие папки (materials/ models/ sound/)
	Скомпилируйте плагин и установите в addons/sourcemod/plugins/
	Перезапустите сервер или смените карту
	Плагин автоматически обработает все файлы и добавит в таблицу загрузок для скачивания клиентами


Для работы с FastDL:

	Разместите .bz2 архивы на FastDL сервере
	Пример - C:/Users/Admin/Desktop/nginx/fastdl/папка-с-контентом
	Настройте sv_downloadurl в server.cfg:
	sv_downloadurl "http://ваш-fastdl-сервер/путь/"
